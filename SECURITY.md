# Security Policy

## Supported Versions

This repository supports **only the current production deployment** of the IEEE TXST website.

| Version | Supported |
| ------ | --------- |
| Production (main branch) | ✅ |
| Development / feature branches | ❌ |
| Archived or previous commits | ❌ |

Security updates are applied only to the live website and the `main` branch.

---

## Reporting a Vulnerability

If you discover a security issue or vulnerability, please report it responsibly.

### How to Report
- Email: **texasstateieee@gmail.com**
- Subject line: **Security Vulnerability – IEEE TXST Website**

Please include:
- A clear description of the issue
- Steps to reproduce (if applicable)
- Screenshots or logs (if helpful)

### What to Expect
- We will acknowledge receipt within **3–5 days**
- Valid issues will be investigated and fixed as soon as possible
- You will be notified once the issue is resolved

> Please do **not** publicly disclose security issues until we have had a chance to address them.

---

## Scope

This policy applies to:
- https://ieeetxst.org
- The code in this GitHub repository

It does **not** cover third-party services such as:
- Google Forms
- Discord
- External event platforms
- IEEE global websites
